# 配置邮件服务器

目前我们接入了第三方发件服务，需要自行注册后获取相关的 `id` 和 `secret`，并填入环境变量。

## Mailgun

| Mailgun 中的名称 | 填入环境变量的名称         | 例子               |
|--------------|-------------------|------------------|
| 邮箱域名         | EMAIL_DOMAIN      | mail.example.com |
| 访问密钥 ID      | SMS_ACCESS_KEY_ID | ffffffffffffffff |
